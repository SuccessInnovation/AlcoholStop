# 절주온 클론코딩 프로젝트

### 프로젝트 소개

본 저장소는 [절주온 공식 홈페이지](https://www.khepi.or.kr/alcoholstop) 메인 화면을 클론 코딩한 프로젝트입니다.  
절주온 서비스의 UI를 참고하여 주요 기능을 재현했으며, 학습 목적을 위해 제작되었습니다.
